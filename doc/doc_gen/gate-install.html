Install

To Install the Textalytics plugin in GATE:

  1. Click File > Manage CREOLE Plugins... .
  2. Add the Plugin URL: http://textalytics.com/files/download/GATE/Textalytics .
  3. Click in Click the Load Now or/and Load Always check.
  4. Once loaded, you can add Textalytics APIs as Processing Resources in the left pane.
